# EdgeRemover

**A professional, silent Microsoft Edge removal tool for Windows**  
Built in C# – Fast, clean, and user-friendly.

![EdgeRemover Screenshot](https://via.placeholder.com/800x450?text=EdgeRemover+Running) <!-- Replace with your screenshot -->

## Features
- Automatic administrator elevation (UAC prompt on launch)
- Beautiful console interface with progress indicators
- Completely silent background operations (no flashing windows)
- Optional WebView2 removal (`--webview` flag)
- Thorough cleanup: files, registry, services, tasks, shortcuts

## Download
Latest version: [EdgeRemover v1.0.0](https://github.com/YOURUSERNAME/EdgeRemover/releases/latest)

## How to Use
1. Download `EdgeRemover.exe` from Releases
2. Double-click → Click **Yes** on UAC prompt
3. Tool runs silently → Restart PC when done

**Advanced**: Run in CMD as admin with `--webview` to remove WebView2 too.
